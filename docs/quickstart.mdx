## Run with Docker

The quickest way to run Proxy Hopper is with the pre-built Docker image:

```bash
docker pull ghcr.io/cams-data/proxy-hopper:latest
```

For the Redis backend (multi-instance HA deployments):

```bash
docker pull ghcr.io/cams-data/proxy-hopper:latest-redis
```

<Note>
  PyPI packages are not yet published. Docker is the recommended installation method. See [Deployment](/deployment/docker) for full Docker and Compose examples.
</Note>

## Write a config file

Create a `config.yaml`. The minimal config needs at least one target:

```yaml
targets:
  - name: general
    regex: '.*'
    ipList:
      - "10.0.0.1:3128"
      - "10.0.0.2:3128"
      - "10.0.0.3:3128"
    minRequestInterval: 1s
    maxQueueWait: 30s
    numRetries: 3
    ipFailuresUntilQuarantine: 5
    quarantineTime: 2m
```

For a config with named proxy providers and shared IP pools:

```yaml
proxyProviders:
  - name: my-provider
    auth:
      type: basic
      username: user
      password: secret
    ipList:
      - "10.0.0.1:3128"
      - "10.0.0.2:3128"
      - "10.0.0.3:3128"
    regionTag: US-East

ipPools:
  - name: my-pool
    ipRequests:
      - provider: my-provider
        count: 3

targets:
  - name: general
    regex: '.*'
    ipPool: my-pool
    minRequestInterval: 1s
    maxQueueWait: 30s
    numRetries: 3
    ipFailuresUntilQuarantine: 5
    quarantineTime: 2m
```

## Start the server

```bash
docker run -v $(pwd)/config.yaml:/etc/proxy-hopper/config.yaml \
  -p 8080:8080 \
  ghcr.io/cams-data/proxy-hopper:latest
```

## Send a request

<CodeGroup>

```bash HTTP proxy mode
curl --proxy http://localhost:8080 https://example.com
```

```bash Forwarding mode
curl -H "X-Proxy-Hopper-Target: https://example.com" \
     http://localhost:8080/api/endpoint
```

```python requests (HTTP proxy)
import requests
resp = requests.get(
    "https://example.com",
    proxies={"https": "http://localhost:8080"}
)
```

```python requests (forwarding mode)
import requests
session = requests.Session()
session.headers["X-Proxy-Hopper-Target"] = "https://example.com"
resp = session.get("http://localhost:8080/api/endpoint")
```

```python aiohttp
import aiohttp
async with aiohttp.ClientSession() as session:
    async with session.get(
        "https://example.com",
        proxy="http://localhost:8080"
    ) as resp:
        print(resp.status)
```

</CodeGroup>

## Validate a config file

```bash
docker run -v $(pwd)/config.yaml:/etc/proxy-hopper/config.yaml \
  ghcr.io/cams-data/proxy-hopper:latest \
  proxy-hopper validate --config /etc/proxy-hopper/config.yaml
# Config OK — 1 target(s) defined.
#   'general': 3 IP(s), regex='.*'
# Server defaults: host=0.0.0.0, port=8080, backend=memory
```
